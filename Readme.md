# 🚀 AstroNet - Conectividade via Satélite | Global Solution

Projeto desenvolvido como solução para a **Global Solution** na disciplina de **Front-End Design Engineering** da **FIAP**. A plataforma web tem como objetivo apresentar a proposta tecnológica da AstroNet, focada em fornecer internet, monitoramento climático e dados em regiões remotas de difícil acesso.

---

## 📌 Sobre o Projeto

A **AstroNet** é uma solução de infraestrutura satelital desenhada para mitigar a exclusão digital global. Ela provê conectividade estável, processamento de dados climáticos e suporte analítico para comunidades e empresas situadas em ecossistemas isolados (áreas rurais, florestas e zonas sem rede de telecomunicação tradicional).

A aplicação web expõe os pilares do ecossistema:
- **Inclusão Digital:** Acesso democrático à internet para saúde, educação e trabalho.
- **Monitoramento Climático:** Painel intuitivo com informações ambientais detalhadas organizadas de forma semanal para a tomada de decisões preventivas.
- **Venda de Dados Estratégicos:** Fornecimento de métricas consolidadas para corporações com operações remotas.
